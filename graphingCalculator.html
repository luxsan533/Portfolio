Graphing Calculator

Graph functions and save them

Overview

This is a full stack website that lets you create graphs out of functions and save them to your account. You can then revist your saved graphs. You can also input values and see the output value from the function.

This website was created with HTML, CSS, Javascript, ExpressJS, MongoDB and Python. Python is used to create the graphs using the matplotlib library. The graphs are sent from python to the javascript file to be displayed on the site.


This was created as a personal project. The code for this project can be found on my github under the repository called 'Graphing Calculator'.

Key Features

  • Create Accounts
  • Write functions to create graphs
  • Input x-values to functions and be given the y-value output
  • Name and save graphs to revisit them
