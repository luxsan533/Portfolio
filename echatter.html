E-chatter

Instant Messaging Social Platform

Overview

This website allows for direct messaging between users. Users can chat through direct messaging to a single user and through servers to multiple users.


This is a full-stack website. The front end was created using React and CSS, while the back end was developed using ExpressJS and MongoDB. Instant messaging is implemented using WebSocket, and the server and client communication is done using Axios.


This was created as a personal project, and the code for it can be found in my GitHub repository, 'E-Chatter'.

Key Features

  • Create Accounts
  • Add Friends
  • Direct Message with Friends
  • Change username and avatar
  • Create or join servers
  • Chat with multiple users in server channels
