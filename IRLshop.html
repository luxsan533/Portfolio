IRL Shop

Simulates an online store

Overview

This is a full stack website created for my web development course. It simulates an online store where users can sell items, search for items and buy them.

The front end was created using HTML, CSS and Javascript while the back-end is created using PHP and MySQL. I created this in a team with 2 other members. The name 'IRL' comes from the first letter of each of our names.


My task was to write the javascript functionality in addition to the php and sql code for the backend, whereas my team members wrote the html and css for all the pages. You can find the code for this project on my github under the repository called 'IRLShop'.

Key Features

  • Create Accounts
  • Search for items
  • Apply filters to searches
  • Add items to shopping cart and wishlist
  • Sell items
  • Check out items
